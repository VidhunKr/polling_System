# 🗳️ Polling System API (Node.js + Express + MongoDB)

This is a simple polling system built using **Node.js**, **Express**, and **MongoDB**. It allows users to:

 Create polls  
 Vote on poll options  
 View all polls and their results

---

## 📦 Technologies Used

- Node.js
- Express
- MongoDB
- Mongoose
- dotenv
- CORS
